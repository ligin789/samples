What is React Native?
React Native is a JavaScript-based framework for building native mobile applications. It allows for the creation of high-performance and responsive mobile apps for iOS and Android platforms using a single codebase.

Why use React Native?
React Native simplifies the app development process by reducing the time and effort required to build high-quality mobile applications. It also provides a range of features that
allow for the creation of beautiful and intuitive user interfaces.

What makes React Native special?
React Native uses a unique architecture allowing it to leverage the power of native components, while still maintaining the speed and flexibility of a JavaScript-based platform.React Native offer collection of react native components
